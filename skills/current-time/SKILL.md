# 获取当前系统时间

Claude 自身无法准确感知当前时间，系统提示中的日期也可能存在偏差。此技能通过运行 Node.js 脚本直接从操作系统获取精确时间，确保返回的时间信息是可靠的。

## 使用方法

运行以下脚本获取当前系统时间：

```bash
node "<skill_path>/scripts/get-time.js"
```

其中 `<skill_path>` 替换为此技能的实际路径。

## 输出格式

脚本输出 JSON 格式的时间信息，包含以下字段：

| 字段 | 示例 | 说明 |
|------|------|------|
| `datetime` | `2026-03-12 14:30:05.123` | 完整日期时间 |
| `date` | `2026-03-12` | 日期 |
| `time` | `14:30:05` | 时间 |
| `weekday` | `星期四` | 星期 |
| `timestamp` | `1773513005123` | Unix 时间戳(毫秒) |
| `iso` | `2026-03-12T06:30:05.123Z` | ISO 8601 格式 |
| `timezone` | `UTC+08:00` | 时区 |

## 回复要求

- 根据用户的问题选择性地返回相关字段，不需要全部展示
- 例如用户问"几点了"，只需回复时间；问"今天几号"，只需回复日期和星期
- 用自然语言回复，不要直接输出 JSON
